Plugin States

A plugin can be in one of these states:

1. Available

Plugin exists in repository but not installed locally.

2. Installed

Plugin binary exists in plugin directory and registered in state.

3. Active

Plugin is loaded by the Lux node and running.

4. Deprecated

Older version installed but newer version available.
